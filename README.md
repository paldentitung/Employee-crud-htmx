# Employee Management App

A simple **Employee Management** web app built with **Node.js**, **Express**, and **HTMX**.  
Allows you to **add, edit, and delete employees** with dynamic UI updates.

---

## Features

- Add new employees via modal form
- Edit existing employees with pre-filled modal form
- Delete employees directly from the table
- Dynamic table updates using **HTMX** (no page reload)

---

## Tech Stack

- **Backend:** Node.js + Express  
- **Frontend:** HTML, CSS, HTMX  
- **Data Storage:** In-memory JavaScript array
